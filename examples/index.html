Ai - Angular Helper Framework

The Ai library is a collection of tools to assist in building Angular applications quickly while maintaining good project structure with simplified syntax. Along with the "app" module various other modules enhance your project with commonly used UI providers and directives.

App Module

The application module initializes your client-side Angular project wiring up areas which contain your controllers, filters, directives, factories and services. Using this syntax simplifies things greatly. The result is a very organized pattern which allows others within your team to follow and understand. 

var modules, options;
modules = [
    'ngRoute'
];
options: {
    ns: 'my_namespace' // default: 'app'
};
$app.init(modules, options);
