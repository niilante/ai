Widgets are a group of helper directives that make working with forms and elements effortless. Widgets have a global provider which allows configuring site wide parameters. This makes it much easier to configure defaults. 

    To configure the widget provider:
    $widgetProvider.set({ // your object of options });

    Or set options for a specific widget.
    $widgetProvider.set('decimal', { // your decimal options });

Decimal

Any numeric value is forced to two decimal places.

Case

Case supports: first, title, upper, lower, pascal, camel.
