The viewer module adds a view slider for views. It works well in mobile environments where simulating a native feel is desired.

There is no example for $viewer as it works with ng-view ONLY!. However the example and options documentation is located below.

  • Markup
  • Code
  • Options
<ai-viewer options="viewerConf"></ai-viewer>

No code is required to use $viewer other than passing your options into the element directive.

// instantiating manually.
$module = $viewer(element, options);

var defaults = {
    template: '<div class="ai-viewer" ng-view />',
    viewCss: 'ai-viewer-view',
    animate: 'slide'
}
