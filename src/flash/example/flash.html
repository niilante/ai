Flash messages are simple pop up messages similar to a modal dialog but with much simpler features. They are designed to display a few sentence or even just a few words.

Flash module can only be instantiated by directive. Flash also is a singleton meaning there is only one instance per application. Flash does though support multiple flashes when required.

Flash messages support 4 parameters. The message, the class type, the title and a timeout. Only the message is required. Flash also supports shifting arguments so you may pass for example a message, type and timeout.

Use the $flashProvider to configure defaults such as the default message type and default timeout for auto removal of messages. The timeout can be specified as false or 0 to disabled (requiring messages too be closed manually) or true can be passed to use the default timeout from the options. You may also pass a specified timeout

angular.module('myModule', [])
.controller('MyController', ['$scope', '$flash', function ($scope, $flash) {

    // $flash - returns the instance of the $flash object.

    // params are
    // message, type(style), title, timeout.
    // setting timeout to 0 sets flash to sticky mode (must be closed by user).
    $flash.add('My flash message', 'success');

    // remove the flash instance.
    $flash.remove(_flash);

}]);
            
var defaults = {
    template: 'flash.tpl.html',             // the template for flash message.
    html: true,                             // when true html flash messages can be used.(requires ngSanitize)
    errors: true,                           // when true flash is shown automatically on http status errors.
    excludeErrors: [401, 403, 404],         // exclude errors by status type.
    errorName: 'Unknown Exception',         // the error name to use in event and error.name is not valid.
    errorMessage: 'An unknown exception ' + // default error message in event one is not provided.
                  'has occurred, if the ' +
                  'problem persists ' +
                  'please contact the ' +
                  'administrator.',
    multiple: false,                        // whether to allow multiple flash messages at same time.
    type: 'ai-flash-info',                  // the default type of message to show also the css class name.
    typeError: 'ai-flash-danger',           // the error type or class name for error messages.
    animation: false,                       // provide class name for animation.
    timeout: 3500,                          // timeout to auto remove flashes after period of time..
                                            // instead of by timeout.
    onError: undefined                      // function called on error before flashed, return false to ignore.

};
